## ⚙️ Setup

1️⃣ **Clone the repo**

git clone https://github.com/<your-username>/assignment2.git
cd assignment2


2️⃣ **Install dependencies**


npm install


3️⃣ **Create model file**


mkdir models


4️⃣ **Add type to package.json**


"type": "module"


5️⃣ **Run the server**

node index.js

Server runs at 👉 [http://localhost:3000](http://localhost:3000)
